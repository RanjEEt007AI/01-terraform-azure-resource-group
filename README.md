# Terraform Azure Resource Group

## Overview

This project demonstrates how to create an Azure Resource Group using Terraform simple.

## Prerequisites

* Azure Subscription
* Terraform Installed
* Azure CLI Installed
* Git

## Project Structure

```text
.
├── main.tf
├── variables.tf
├── .gitignore
└── README.md
```

## Deployment Steps

### 1. Clone Repository

```bash
git clone https://github.com/RanjEEt007AI/01-terraform-azure-resource-group.git
cd 01-terraform-azure-resource-group
```

### 2. Login to Azure

```bash
az login
```

### 3. Initialize Terraform

```bash
terraform init
```

### 4. Validate Configuration

```bash
terraform validate
```

### 5. Create Execution Plan

```bash
terraform plan
```

### 6. Deploy Resources

```bash
terraform apply
```
